# 🎬 Кино.Форум

Кинофорум в стиле Яндекса — сайт для обсуждения фильмов, написания рецензий и общения с другими киноманами.

## 📸 Модель

Главная страница с каталогом фильмов, топом и последними рецензиями.  
Страница фильма с описанием, рейтингом и отзывами.  
Форум с темами и комментариями.  
Профили пользователей с избранными фильмами.

## 🚀 Возможности

- 📝 Регистрация и авторизация пользователей
- 🎥 Каталог фильмов с фильтрацией по жанру, стране, году
- ⭐️ Система рейтинга и оценок фильмов
- 📖 Написание рецензий к фильмам
- 💬 Форум с темами и комментариями
- 🔍 Поиск по фильмам, рецензиям и темам
- ❤️ Избранные фильмы
- 👤 Профили пользователей
- 📞 Форма обратной связи
- 🎨 Дизайн в стиле Яндекса

## 🛠 Технологии

- **Python 3.9+**
- **Flask** — веб-фреймворк
- **Flask-Login** — авторизация
- **Flask-WTF** — формы
- **SQLAlchemy** — работа с базой данных
- **SQLite** — база данных
- **WTForms** — валидация форм
- **Werkzeug** — хеширование паролей
